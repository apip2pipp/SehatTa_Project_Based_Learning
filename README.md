# Avenution 🥗✨

Avenution adalah sistem rekomendasi menu makanan berbasis kondisi tubuh yang dirancang untuk membantu pengguna memilih makanan yang sesuai dengan kebutuhan kesehatan mereka.

Nama Avenution berasal dari gabungan kata "Avenue" (jalan) dan "Nutrition" (nutrisi), yang berarti jalan menuju pola makan yang lebih sehat dan terarah.

Sistem ini menganalisis berbagai parameter kondisi tubuh seperti usia, berat badan, tinggi badan, BMI, riwayat penyakit, serta tujuan kesehatan (diet, menaikkan berat badan, menjaga kesehatan, dll).

Dengan pendekatan sistem rekomendasi berbasis data, Avenution memberikan saran menu makanan yang lebih personal, relevan, dan terarah.

---

## 🎯 Tujuan Sistem

- Membantu pengguna memilih menu makanan yang sesuai dengan kondisi tubuh
- Memberikan rekomendasi berbasis analisis data kesehatan
- Meningkatkan kesadaran pola makan sehat berbasis kebutuhan individu
- Mengembangkan prototype sistem rekomendasi berbasis Artificial Intelligence

---

## ⚙️ Fitur Utama

- Input kondisi tubuh pengguna
- Perhitungan BMI otomatis
- Analisis kebutuhan nutrisi
- Rekomendasi menu makanan personal
- Riwayat pengecekan pengguna
- Dashboard yang user-friendly

---

## 🧠 Metode

Sistem ini menggunakan metode Naive Bayes untuk menghasilkan rekomendasi yang sesuai dengan profil pengguna.

Metode Naive Bayes digunakan untuk:
- Mengklasifikasikan kondisi tubuh pengguna
- Menghitung probabilitas kecocokan menu makanan
- Menghasilkan rekomendasi berdasarkan data yang telah dipelajari

---

## 🏗️ Struktur Halaman (Prototype)

1. Landing Page
2. Login Page
   - User login menggunakan Google Account
   - Admin login menggunakan username & password
3. Dashboard
   - Input data kondisi tubuh
   - Proses analisis
   - Hasil rekomendasi
   - Riwayat hasil pengecekan

---

## 📊 Parameter Analisis

- Usia
- Berat Badan
- Tinggi Badan
- Body Mass Index (BMI)
- Riwayat Penyakit
- Tujuan Kesehatan (Diet, Bulking, Maintain, dll)

---

## 🚀 Teknologi yang Digunakan

- Backend: Laravel
- Database: MySQL
- Frontend: Blade Template
- Metode Machine Learning: Naive Bayes
